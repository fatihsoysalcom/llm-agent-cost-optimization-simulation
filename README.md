# LLM Agent Cost Optimization Simulation

This example simulates an LLM agent workflow to demonstrate how optimizing the number of API calls can reduce costs. It compares an unoptimized approach, making multiple individual LLM calls, with an optimized approach that combines inputs to achieve the same task with fewer calls. The simulation tracks "token costs" to illustrate the potential savings.

## Language

`python`

## How to Run

Save the code as `main.py`. Run from your terminal using `python main.py`.
